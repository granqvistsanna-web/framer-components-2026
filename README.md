## Getting Started

1. **Copy** the code from any component file in the `shipped/` folder
2. In your Framer project, go to **Assets** > **Code** > **New Code File**
3. **Paste** the code and save
4. **Drag** the component onto your canvas
5. **Customize** using the property controls in the right panel — every component works out of the box with sensible defaults

> Some components use external libraries (GSAP, Three.js, Matter.js). These are loaded automatically via CDN — no extra setup needed.

## Tips

- **Every component ships with defaults** — it will look great the moment you drop it on the canvas
- **Resize freely** — components are responsive and adapt to their container
- **Animation controls** are exposed as props so you can fine-tune timing, easing, and speed without touching code
- **Overflow** — some particle/3D components may need overflow set to `visible` on the parent Frame in Framer
